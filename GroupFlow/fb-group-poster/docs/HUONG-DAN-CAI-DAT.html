1Tải extension

GroupFlow có thể được tải từ 2 cách:

Cách A (nếu lên Chrome Web Store):

Truy cập Chrome Web Store, tìm "GroupFlow" → bấm "Add to Chrome"

Hình: Chrome Web Store với GroupFlow v1.0.277

Cách B (Load unpacked từ folder):

Nếu bạn có file groupflow.zip từ tác giả:

  1. Giải nén file groupflow.zip
  2. Nhớ đường dẫn folder groupflow/fb-group-poster/
  3. Tiếp tục bước 2

2Giải nén folder GroupFlow

Nếu bạn tải qua Cách B (từ zip), bạn cần giải nén folder trước khi load unpacked

Trên Windows: Bấm chuột phải vào file zip → chọn "Extract All" → chọn vị trí lưu

Trên Mac/Linux: Double-click file zip → tự động giải nén

Hình: Giải nén file zip

Sau khi giải nén, bạn sẽ có folder: GroupFlow/fb-group-poster/ (chứa manifest.json, background.js, v.v.)

Nhớ vị trí folder này — sẽ dùng ở bước 5 (Load unpacked)

3Vào chrome://extensions → Load unpacked

Bước 3.1: Mở trang Chrome Extensions

Trong Chrome, nhập vào thanh địa chỉ:

chrome://extensions

Hoặc: Menu (⋮) → Công cụ khác → Tiện ích

Hình: Trang Tiện ích của Chrome

Bước 3.2: Bật Developer Mode

Ở góc phải trên của trang, bạn sẽ thấy công tắc "Developer mode"

Bấm công tắc này để bật Developer mode (công tắc sẽ chuyển sang xanh)

🇬🇧 Enable Developer mode | 🇻🇳 Bật chế độ Nhà phát triển (Developer mode)

ℹ️ Lưu ý: Developer mode cho phép cài đặt extension từ máy tính mà không cần qua Chrome Web Store.

Bước 3.3: Bấm Load unpacked

Sau khi bật Developer mode, 3 nút mới sẽ xuất hiện ở đầu trang:

  • Load unpacked — cài đặt extension từ folder
  • Pack extension
  • Update extensions now

Bấm nút "Load unpacked"

🇬🇧 Click "Load unpacked" button | 🇻🇳 Bấm nút "Load unpacked"

Bước 3.4: Chọn folder fb-group-poster

Một hộp thoại chọn folder sẽ mở ra

Điều hướng tới folder GroupFlow/fb-group-poster/ (nơi chứa manifest.json)

Bấm "Select Folder" (hoặc tương tự tùy hệ điều hành) → GroupFlow sẽ cài đặt thành công

✅ GroupFlow đã cài xong! Bạn sẽ thấy icon GroupFlow (chữ G xanh) ở góc trên phải Chrome

4Nhập License Key từ tidien.xyz

Để hoạt động đầy đủ, GroupFlow cần License Key từ tidien.xyz

Bước 4.1: Đăng nhập tidien.xyz

Truy cập https://tidien.xyz/user/login

Nhập Email + Mật khẩu của bạn → bấm "Đăng nhập"

Bước 4.2: Lấy License Key

Sau khi đăng nhập, bạn sẽ thấy Dashboard

Tìm mục "License Key" → bấm nút "Copy" để sao chép key

🇬🇧 Load unpacked button + tidien.xyz License Key | 🇻🇳 Nút Load unpacked + Lấy License Key từ tidien.xyz

Bước 4.3: Nhập vào GroupFlow

Quay lại Chrome, bấm icon GroupFlow (ở góc trên phải)

Vào tab "Cài đặt" → "Extension"

Dán License Key vừa copy vào ô "License Key"

Bấm "Xác thực key"

🇬🇧 Enter License Key in GroupFlow | 🇻🇳 Nhập License Key vào GroupFlow extension

✅ Hoàn thành! GroupFlow đã sẵn sàng sử dụng. Bạn có thể bắt đầu tạo bài, comment chéo, lên lịch, v.v.

❓ FAQ & Khắc phục sự cố

Làm sao để biết GroupFlow đã cài thành công?
Khi cài xong, bạn sẽ thấy icon GroupFlow (chữ "G" xanh) ở góc trên phải Chrome. Bấm vào → panel GroupFlow mở ra với các tab: Tạo bài, Comment, Cài đặt, v.v.
Panel GroupFlow không mở, chỉ thấy icon nhưng bấm không có gì?
Cách 1: Reload extension — vào chrome://extensions, tìm GroupFlow → bấm icon ↻ (refresh)

Cách 2: Thử mở Facebook.com trước, rồi bấm icon GroupFlow (extension chỉ hoạt động trên Facebook)

Cách 3: Mở Console (F12) → Console tab → Tìm lỗi đỏ, báo cho tác giả
Lỗi "License Key không hợp lệ"?
Kiểm tra:
1. Key có bị cắt (copy dỏm) không? → Copy lại từ tidien.xyz Dashboard
2. Có khoảng trắng thừa không? → Xoá các khoảng trắng đầu/cuối
3. Key đã hết hạn không? → Vào tidien.xyz xem "Hoạt động cuối" — nếu hết hạn, liên hệ tác giả để gia hạn
Có cách nào xoá/gỡ GroupFlow không?
Vào chrome://extensions → Tìm GroupFlow → bấm nút "Remove" (hoặc tương tự) → Xác nhận
GroupFlow có hoạt động offline không?
Không. GroupFlow cần kết nối internet để:
- Đồng bộ bài viết với tidien.xyz
- Gọi API Facebook để đăng bài / comment
- Dùng AI text/image generation
Nếu mất mạng, bạn vẫn có thể tạo bài cục bộ, nhưng không thể đồng bộ hay đăng cho tới khi có mạng trở lại.
Tôi có 5 máy khác nhau, cài GroupFlow trên tất cả được không?
Được. GroupFlow hỗ trợ multi-device sync — một License Key có thể dùng trên nhiều máy cùng lúc. Bạn sẽ thấy danh sách "Devices" ở tidien.xyz để quản lý.
Có bước nào là nguy hiểm / có thể làm hỏng máy không?
Không. GroupFlow chỉ là extension Chrome, hoạt động trong trình duyệt. Nó không có quyền:
- Truy cập files hệ thống
- Sửa đổi Windows/macOS/Linux
- Cài đặt phần mềm khác
- Chạy mã độc
An toàn để cài đặt và sử dụng.
