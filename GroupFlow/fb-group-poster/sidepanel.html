Hướng dẫn sử dụng

Mục tiêu: tạo bài → chọn nhóm → (xuất ảnh nếu cần) → lên lịch/đăng ngay → sync về website.

1) Cài đặt lần đầu (5 phút)

  1. Load unpacked thư mục GroupFlow/fb-group-poster (không chọn ref-group-posting).
  2. Tab Cài đặt → khai báo AI Provider local + chọn thư mục lưu ảnh.
  3. Tab Skill → import skill JSON (tuỳ chọn).
  4. Đăng nhập tidien chỉ khi cần Tải web / sync.

Nếu chưa có provider: tạo trên website tại trang Providers (giống cấu hình fanpage).

2) Quy trình đăng bài

A. Tạo bài
  • Tab Tạo bài → Nhập tay, AI viết hoặc Excel.
  • Nhập nội dung (A–D) + prompt ảnh (nếu cần).
  • Tick bài → Sửa: đổi nội dung, lịch, nhóm đăng trong cùng form; hoặc Chọn nhóm trên card khi không sửa.
B. Chuẩn bị media
  • Đính kèm ảnh/video thủ công, hoặc bấm Generate để xuất ảnh AI.
  • Bật Tự xuất ảnh khi đăng nếu muốn để tới giờ đăng mới xuất ảnh.
C. Đăng ngay / Lên lịch
  • Tick bài cần chạy → bấm Đăng ngay hoặc Lên lịch.
  • Sau đăng OK: card ✓ Đã đăng → Mở bài (comment tay) hoặc ▶ Bot.
  • Bật First comment (bước 3) để bot tự comment ngay sau đăng.
  • Nút Dàn: lên lịch nhiều bài theo khoảng cách phút (campaign).
  • Tab Log: xem lịch sử; tab Activity: quản lý job sắp chạy.

3) Comment chéo team

  • Tab Comment → bấm Làm mới để tải danh sách cần comment.
  • Bấm AI để generate comment (dùng Text Provider) → Chạy đã chọn hoặc Lên lịch.

4) Lỗi thường gặp

  • Nút “Lỗi” đỏ / không load: kiểm tra không copy nhầm thư mục _ref-* hoặc 2.3.2_0 vào folder extension → Reload.
  • Không thấy nhóm: đăng nhập Facebook trong Chrome → vào tab Nhóm → ↻ Làm mới.
  • Generate/AI báo thiếu key: vào Cài đặt → đăng nhập tidien → chọn provider; 9Router key chỉ là dự phòng.
  • Video không đăng: chuyển Chế độ Cổ điển (Settings) vì video cần DOM.
  • Lỗi Nhanh / mở FB bất ngờ: mặc định Nhanh không mở FB; nếu lỗi xem đúng dòng đỏ trong Lịch sử (vd. thiếu session → mở facebook.com F5). Chỉ bật「Nhanh lỗi → Cổ điển」khi cần dự phòng DOM.
